# modul151

Neue Benutzer können sich über ein Formular registrieren.
Nach dem Login können die angemeldeten Benutzer neue Beiträge erstellen oder Kommentare zu Beiträgen anderer Benutzer schreiben. Beiträge können gelöscht und Beschreibungen editiert werden.
Alle Benutzereingaben werden client- und serverseitig Validiert. 
Des Weiteren werden schützenswerte Benutzerdaten sicher in der Datenbank abgelegt. 
Im Administrationsbereich arbeitet die Applikation mit Sessions.
SQL-Injection, Script-Injection und Session-Hijacking werden mit geeigneten Methoden verhindert.
Der Benutzer wird durch Meldungen durch die Applikation geführt.
